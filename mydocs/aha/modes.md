Available Commands:
  /mode        - Set or toggle a mode (e.g., /mode plan)
  /modes       - List available modes
  /save        - Save conversation transcript
  /status      - Show session status
  /clear       - Clear conversation context
  /help        - Show available commands
  /config      - Show current configuration
  /tools       - List available tools
  /agents      - List available agents
  /allowed-dirs - Manage allowed write directories
  /denied-dirs - Manage denied write directories
  /rename      - Rename current session
  /fork        - Fork session at turn N: /fork

Mode Shortcuts:
  /admissions  - Evaluate whether a problem is suitable for an autonomous development machine
  /brainstorm  - Design refinement before any creative work - explore approaches and trade-offs through collaborative dialogue
  /careful     - Full capability with confirmation for destructive actions
  /debug       - Systematic 4-phase debugging - root cause before fixes, evidence before claims
  /execute-plan - Execute implementation plan using subagent-driven development with two-stage review
  /explore     - Zero-footprint exploration - understand before acting
  /finish      - Complete development work - verify tests, present merge/PR/keep/discard options, clean up
  /generate-machine - Generate development machine artifacts into your project
  /machine-design - Design an autonomous development machine for your project (founding session)
  /plan        - Analyze, strategize, and organize - but don't implement
  /verify      - Evidence-based completion verification - no claims without fresh proof
  /write-plan  - Create detailed implementation plan with bite-sized TDD tasks - complete code, exact paths, zero ambiguity
